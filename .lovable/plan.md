# Одностраничник: результаты 1-й недели пилота ИРБ

Цель — наглядно показать клиенту эффект первой недели пилота (неделя 22 июня) на фоне 5 предыдущих недель его собственной работы и контрольной группы. Без вопросов «а точно ли это сработало».

## Структура страницы (одна прокручиваемая, без переходов)

1. **Hero / шапка**
   - Заголовок: «Пилот лидогенерации — итоги 1-й недели»
   - Подзаголовок: период 22 июня, периметр МСК+СПб и 10 регионов, контрольная группа.
   - 4 ключевых KPI-карточки (большими цифрами):
     - Лиды всего пилота: **831** (+23,66% н/н, +41,18% к среднему за 5 нед.)
     - CPL весь пилот: **170 ₽** (−25,12% н/н, −10,05% к среднему за 5 нед.)
     - Лиды регионы: **667** (+28,27% н/н, +41,73% к ср. 5 нед.)
     - CPL регионы — сознательный рост к среднему (+42,64%) при росте объёма → пояснение тут же.

2. **Пилот vs Контроль — главный блок-доказательство**
   - Две колонки сравнения: «Пилот» / «Контрольная группа» с дельтами:
     - Рост лидов: **+41,18%** vs **+20,84%** (5 нед.) — пилот в 2× быстрее.
     - Рост лидов (3 нед.): **+24,46%** vs **+13,05%**.
     - Изменение CPL: **−10,05%** vs **−25,92%** — комментарий: контрольная сокращала объём, пилот наращивал.
   - Вывод-плашка: «Пилот даёт двукратный рост лидов относительно контрольной группы».

3. **Динамика по неделям — графики (Recharts)**
   - График 1: Лиды по неделям (линии: МСК+СПб, Регионы, Весь пилот, Контрольная) — 6 точек: 18.05 → 22.06. Последняя точка выделена (старт пилота).
   - График 2: Бюджет по неделям (те же серии).
   - График 3: CPL по неделям (те же серии).
   - Под каждым графиком короткий текст-инсайт.

4. **Разбивка по сегментам (таблица-карточки)**
   Три карточки: МСК+СПб / Регионы / Весь пилот. В каждой:
   - Лиды (неделя пилота, дельта н/н, дельта к ср. 5 нед., дельта к ср. 3 нед.)
   - Бюджет
   - CPL и его изменение
   - Короткий вывод (напр.: «МСК+СПб: лиды +7,89%, CPL −46,67% — резкое удешевление лида»).

5. **Итоги первой недели** (буллеты):
   - Рост лидов в 2× быстрее, чем у контрольной группы.
   - CPL по МСК+СПб снижен на 46,67% за неделю.
   - В регионах нарастили объём (+28% лидов) с прогнозируемым ростом CPL — масштабирование.
   - Общий бюджет пилота даже снизился (−7,4%) при росте лидов.

6. **Футер**: период данных, методология (что считаем неделей, что такое контрольная группа), контакт.

## Интерактив
- Переключатель сегмента (МСК+СПб / Регионы / Весь пилот / Контрольная) над графиками — фильтрует активные серии.
- Тогглы метрики на одном графике: Лиды / Бюджет / CPL.
- Tooltip Recharts с форматированием ₽ и %.
- Подсветка точки «22 июня» вертикальной референс-линией «Старт пилота».
- Плавное появление секций при скролле (framer-motion / простой intersection).

## Технические детали
- Один маршрут `/` (`src/routes/index.tsx`), заменить плейсхолдер.
- Все данные — константа в `src/lib/pilot-data.ts` (из скриншота, RU-локаль).
- Графики: `recharts` (`bun add recharts`).
- Стили: существующие токены Tailwind v4 + shadcn Card/Badge/Tabs. Без хардкода цветов — добавлю несколько семантических токенов (success / warning / pilot / control) в `src/styles.css`.
- Числа форматируются через `Intl.NumberFormat('ru-RU')`, проценты — со знаком.
- Мета: title/description/og под «Пилот ИРБ — итоги 1-й недели».
- Адаптив: десктоп — 12-колоночная сетка, мобайл — карточки в столбик.
